---

## 🧠 1️⃣ 核心状态（输入）

```python
state = {
    "mid": mid_price,
    "vol": sigma,
    "inventory": q,
    "OFI": order_flow_imbalance,
    "spread": current_spread,
    "k": liquidity_K
}
```

---

## 🧠 2️⃣ 核心信号（alpha）

👉 来自论文3 + 论文10 + GM思想

```python
alpha = w1 * OFI + w2 * short_term_trend
```

---

## 🧠 3️⃣ 动态参数（论文1 + 论文2）

```python
gamma = RL_model(state)          # 风险厌恶（动态）
K = update_K_MLE(trade_data)     # 流动性学习
```

---

## 🧠 4️⃣ Reservation Price（核心）

👉 AS + 趋势 + 多资产思想

```python
r = mid + alpha - gamma * sigma**2 * q
```

---

## 🧠 5️⃣ Spread（成交 + 风险）

```python
spread = gamma * sigma**2 + 1/K
```

---

## 🧠 6️⃣ 最终报价

```python
bid = r - spread/2
ask = r + spread/2
```

---

## 🧠 7️⃣ 动态 skew（订单流驱动）

```python
if OFI > 0:
    bid += skew
if OFI < 0:
    ask -= skew
```

---

## 🧠 8️⃣ 成交概率模型（论文3核心🔥）

```python
fill_prob = exp(-K * delta + beta * OFI)
```

---

## 🧠 9️⃣ 风控（必须）

```python
if abs(q) > q_limit:
    reduce_position()

if vol_spike:
    widen_spread()

if trend_extreme:
    stop_mm()
```
