# 📄 一、系统说明

本系统为基于市场微结构理论的高频做市系统，核心融合了经典做市模型与订单流信息，具备可扩展至实盘交易的工程能力。

一、模型基础  
系统以 Avellaneda–Stoikov（AS）模型为核心，通过库存风险约束构建最优报价结构：

r = mid + alpha − γσ²q  
spread = γσ² + 1/K

其中：  
mid：中间价  
alpha：短期价格信号  
γ：风险厌恶系数  
σ：波动率  
q：库存  
K：流动性参数

二、信号来源（Alpha）  
系统引入订单流不平衡（Order Flow Imbalance, OFI）与短期趋势信号：

alpha = w₁·OFI + w₂·trend

OFI刻画市场主动买卖强度，trend反映短期价格动量。

三、成交建模  
采用指数型成交概率模型：

P(fill) = exp(-K·Δ + β·OFI)

用于刻画报价距离与订单流对成交概率的影响。

四、策略逻辑  
系统通过动态调整买卖报价（bid/ask）实现：

1）赚取买卖价差（spread收益）  
2）利用订单流信号获取方向性收益（alpha收益）  
3）通过库存惩罚项控制风险

五、风控机制  
系统内置多层风险控制：

- 库存限制（Inventory Limit）
    
- 波动率过滤（Volatility Filter）
    
- 单边行情暂停（Kill Switch）
    

六、系统特点

1）基于市场微结构（非纯预测模型）  
2）具备实时动态调整能力  
3）支持扩展至强化学习（RL）优化参数  
4）具备实盘部署潜力

七、收益来源分解

PnL = spread收益 + alpha收益 − inventory风险 − adverse selection

本系统属于“库存驱动 + 信息驱动”的混合做市模型。

## 📌 core/engine.py（核心引擎）

```python
import numpy as np


class MMEngine:
    """
    ==============================
    Market Making Engine（核心做市引擎）
    ==============================

    参数说明（非常重要）：

    gamma (float):
        风险厌恶系数（Inventory Risk Aversion）
        - 越大 → 越保守 → spread更宽 → 库存更安全
        - 越小 → 越激进 → 更容易赚钱但风险更高
        建议范围：0.05 ~ 0.3

    k_init (float):
        初始流动性参数（Liquidity Parameter）
        - 表示市场成交难度
        - 越大 → 越容易成交
        - 会被动态更新（核心）

    w_ofi (float):
        OFI权重（订单流信号权重）
        - 控制订单流对价格的影响
        - 建议：0.5 ~ 1.0

    w_trend (float):
        趋势权重（短期价格动量）
        - 辅助alpha信号
        - 建议：0.0 ~ 0.5

    ==============================
    """

    def __init__(self, config):
        self.gamma = config["gamma"]
        self.k = config["k_init"]
        self.w_ofi = config["w_ofi"]
        self.w_trend = config["w_trend"]

    # ==============================
    # 流动性参数 K（核心）
    # ==============================
    def update_k(self, trades, mid):
        """
        K = 1 / E[|trade_price - mid|]

        含义：
        - 距离mid越远还能成交 → 市场流动性强 → K大
        - 距离很近才成交 → 流动性弱 → K小
        """

        if len(trades) < 20:
            return self.k

        deltas = np.array([abs(t["price"] - mid) for t in trades[-200:]])
        mean_delta = np.mean(deltas)

        self.k = 1.0 / (mean_delta + 1e-6)
        self.k = np.clip(self.k, 0.01, 50)

        return self.k

    # ==============================
    # Alpha（信号）
    # ==============================
    def compute_alpha(self, ofi, trend):
        """
        alpha = w1 * OFI + w2 * trend

        OFI：订单流不平衡
        trend：短期价格变化

        alpha > 0 → 看涨
        alpha < 0 → 看跌
        """
        return self.w_ofi * ofi + self.w_trend * trend

    # ==============================
    # 核心报价函数
    # ==============================
    def quote(self, state):
        """
        输入：
            state = {
                mid, vol, inventory, ofi, trend
            }

        输出：
            bid, ask, spread 等
        """

        mid = state["mid"]
        sigma = state["vol"]
        q = state["inventory"]
        ofi = state["ofi"]
        trend = state["trend"]

        # === alpha ===
        alpha = self.compute_alpha(ofi, trend)

        # === Reservation Price（核心） ===
        # r = mid + alpha - gamma * sigma^2 * q
        r = mid + alpha - self.gamma * (sigma ** 2) * q

        # === Spread ===
        # spread = gamma * sigma^2 + 1/K
        spread = self.gamma * (sigma ** 2) + 1.0 / self.k

        # === 报价 ===
        bid = r - spread / 2
        ask = r + spread / 2

        # === OFI skew（订单流偏移） ===
        skew_strength = 0.15  # 控制方向偏移强度
        if ofi > 0:
            bid += spread * skew_strength
        elif ofi < 0:
            ask -= spread * skew_strength

        return {
            "bid": float(bid),
            "ask": float(ask),
            "mid": float(mid),
            "spread": float(spread),
            "alpha": float(alpha),
            "k": float(self.k)
        }

    # ==============================
    # 成交概率模型
    # ==============================
    def fill_probability(self, delta, ofi):
        """
        P(fill) = exp(-K * delta + beta * OFI)

        delta：报价距离mid
        OFI：订单流

        用于模拟成交概率（回测/仿真）
        """
        return float(np.exp(-self.k * delta + 0.5 * ofi))
```

## 📌 features（信号层）

```python
import numpy as np


def compute_ofi(orderbook):
    """
    OFI（订单流不平衡）

    使用前5档盘口
    """
    bids = orderbook["bids"][:5]
    asks = orderbook["asks"][:5]

    bid_vol = sum([b[1] for b in bids])
    ask_vol = sum([a[1] for a in asks])

    return (bid_vol - ask_vol) / (bid_vol + ask_vol + 1e-6)


def compute_volatility(prices):
    """
    波动率（log return std）
    """
    if len(prices) < 50:
        return 0

    returns = np.diff(np.log(prices))
    return float(np.std(returns[-100:]))


def compute_trend(prices):
    """
    短期趋势（简单动量）
    """
    if len(prices) < 10:
        return 0

    return float(prices[-1] - prices[-5])
```

## 📌 risk（风控）

```python
class RiskManager:
    """
    风控系统
    """

    def __init__(self, max_inventory=0.02, max_vol=0.02):
        self.max_inventory = max_inventory
        self.max_vol = max_vol

    def check(self, state):
        q = state["inventory"]
        vol = state["vol"]

        if abs(q) > self.max_inventory:
            return "HEDGE"

        if vol > self.max_vol:
            return "WIDEN"

        return "NORMAL"
```

## 📌 run.py（主入口）

```python
engine = MMEngine({
    "gamma": 0.1,
    "k_init": 1.0,
    "w_ofi": 0.7,
    "w_trend": 0.3
})

risk = RiskManager()

prices = []

while True:
    orderbook = get_orderbook()
    trades = get_trades()
    inventory = get_position()

    best_bid = orderbook["bids"][0][0]
    best_ask = orderbook["asks"][0][0]
    mid = (best_bid + best_ask) / 2

    prices.append(mid)

    ofi = compute_ofi(orderbook)
    vol = compute_volatility(prices)
    trend = compute_trend(prices)

    engine.update_k(trades, mid)

    state = {
        "mid": mid,
        "vol": vol,
        "inventory": inventory,
        "ofi": ofi,
        "trend": trend
    }

    quote = engine.quote(state)

    action = risk.check(state)

    if action == "HEDGE":
        hedge_position()
        continue

    if action == "WIDEN":
        quote["bid"] -= 1
        quote["ask"] += 1

    place_orders(quote["bid"], quote["ask"])
```
