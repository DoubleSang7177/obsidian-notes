## 道士理論123法則--用於判斷趨勢

![](file:////Users/sang/Library/Containers/com.kingsoft.wpsoffice.mac.global/Data/tmp/wps-sang/ksohtml//wps34.jpg) 

![](file:////Users/sang/Library/Containers/com.kingsoft.wpsoffice.mac.global/Data/tmp/wps-sang/ksohtml//wps35.jpg) 

上涨趋势线的组成：上涨趋势为不断抬高的低点（HL），以及不断抬高的高点（HH）组成

下跌趋势线的组成：下跌趋势为不断降低的高点（LH），以及不断降低的低点（LL）组成

1 为突破/跌破趋势线

2 为回测无法创造新低/新高

3 为再创新高/新低。 突破前高/前低

用于趋势判断-趋势终结、趋势反转

趋势终结：123法则成立后原趋势终结，趋势终结可能会走盘整，盘整之后有机会延续原趋势也有可能趋势反转

趋势反转：123法则成立后趋势直接反转，通常1和3为大动能时，趋势反转的机率较高

# ETF

## 期貨vs現貨

現金交割和實物兩種。期貨的買賣不會直接觸發現貨的交易，只有在現貨和期貨出現不較大的價格差的時候，才會觸發一些現貨市場的買賣。所以說發100個btc期貨ETF也只會買期貨ETF這種場外對賭產品，所以對btc現貨惡價格會有影響但是這種影響是間接的；發現貨ETF就會直接影響影響現貨價格，大量邁入會產生直接的拉盤效果

## 概念

Exchange-Traded Found 交易所交易基金，是一種在證券交易所上市交易的開放式基金，結合了股票和共同基金的特點。完全可以在二級市場上像買賣股票一樣區交易，隨時買賣，邁入賣出價可能不完全匹配這個持倉價值（正常的基金應該是有持倉時間要求的）

# 经济数据

## 初请失业金人数

**概念**：初次申请失业保险金的人数，是反映美国劳动力市场健康状况的最及时、最敏感的指标之一。

**人数下降**：通常表明就业市场强劲，经济向好，可能利好美元，但可能令股市（担心加息）承压。

**人数上升**：通常表明就业市场疲软，经济可能放缓，可能利空美元，但可能提振股市（期待宽松政策）。

**1****. 对美元（USD）的影响**

利好美元：初请人数 < 预期且下降

逻辑：就业市场强劲 → 工资和消费有支撑 → 通胀压力可能持续 → 美联储更有理由维持高利率或放缓降息步伐 → 美元吸引力增强。

利空美元（美元指数下跌）：初请人数 > 预期且上升：

逻辑：就业市场出现裂痕 → 经济可能放缓 → 通胀压力减轻 → 市场强化对美联储提前/加快降息的预期 → 美元利率优势减弱。

2. **对贵金属（黄金、白银）的影响**

核心驱动力：实际利率预期（= 名义利率 - 通胀预期）和美元。黄金不生息，与美元通常呈负相关。

数据表现好（利空黄金）：

强化加息/维持利率预期 → 实际利率预期上升 + 美元走强 → 双重利空黄金，金价承压。

数据表现差（利好黄金）：

强化降息预期 → 实际利率预期下降 + 美元走弱 → 双重利好黄金，金价上涨。

若数据极差引发避险情绪，黄金的避险属性也会得到支撑

3. **对美股（标普500、纳斯达克等）的影响****：双面**

利好：初请人数下降：

短期矛盾：

积极面：经济基本面健康，利于企业盈利前景，支撑股市。

消极面：可能推迟美联储降息，高利率环境压制股市估值（尤其是对利率敏感的科技成长股）。

结果：影响复杂，取决于市场当前的主导逻辑。在经济担忧时期，“好消息就是好消息”；在通胀担忧时期，“好消息（对美联储而言）可能就是坏消息（对股市而言）”。

利空：初请人数上升：

短期矛盾：

消极面：引发对经济衰退和企业盈利的担忧。

积极面：大幅提升对美联储降息的预期，流动性前景改善，提振市场情绪和估值。

结果：同样复杂。但若数据仅是温和走弱，未引发衰退恐慌，市场通常更倾向于交易“降息预期”，从而推动股市上涨。

4. **对风险资产（以比特币为代表）的影响****： 双面**

核心驱动力：全球流动性预期和风险偏好。比特币等加密资产被视为“高贝塔”的风险资产。

数据表现好（初请人数下降）：

若市场解读为“经济健康，无衰退风险”，可能提振风险偏好，对比特币略偏正面。

若市场解读为“美联储将更鹰派，收紧流动性”，则利空比特币（流动性收紧不利于所有风险资产）。

数据表现差（初请人数上升）：

这是更清晰、更主要的传导路径：数据差 → 强化降息/宽松预期 → 全球流动性前景改善 → 风险资产（包括股市、加密货币）获得流动性注入的预期 → 通常利好比特币等加密货币。

比特币近年来与科技股（尤其是纳斯达克）的相关性较高，常跟随类似的宏观逻辑。
